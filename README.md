# Лабораторная работа 10: Сравнительный анализ архитектур глубоких нейронных сетей для сегментации изображений

## 🎯 Цель работы
Изучить современные архитектуры глубоких нейронных сетей для задачи семантической сегментации изображений, провести сравнительный анализ их эффективности, исследовать влияние гиперпараметров на качество обучения и освоить методы оптимизации вычислительных ресурсов при работе с большими наборами данных.

---

## 🛠 Технологический стек
- **Framework:** TensorFlow 2.x (Keras), PyTorch 2.x
- **Data Processing:** NumPy, scikit-learn, PIL
- **Visualization:** Matplotlib, Seaborn
- **Hardware:** T4 GPU, 15 GB VRAM

---
## 📊 Структура и задачи исследования

### Задание 1. Исследование влияния гиперпараметров на U-Net
На основе учебного ноутбука проведите финальную подготовку данных. Измените количество сегментирующих классов с 16 на 5. Проведите суммарно не менее 10 экспериментов и визуализируйте их результаты (включая точность обучения сетей на одинаковом количестве эпох, например, на 7), изменив:
- количество фильтров в сверточных слоях
- размер ядра в сверточных слоях
- функцию активации в скрытых слоях

### Задание 2. Реализация U-Net с skip connections
На основе учебного ноутбука проведите финальную подготовку данных. Измените количество сегментирующих классов с 16 на 7. Изучите внимательно особенности U-Net, определите принципиальное отличие U-Net и simpleUnet из учебного ноутбука. Доработайте simpleUnet с учетом особенностей U-Net. Обучите модель на 100 эпохах и визуализируйте результат.

### Задание 3. Реализация PSPNet с Pyramid Pooling Module
На основе учебного ноутбука проведите финальную подготовку данных. Измените количество сегментирующих классов с 16 на 7. Реализуйте сегментацию базы Стройка на основе модели PSPNet, изучив статью Pyramid Scene Parsing Network.

## 📈 Ключевые результаты

| Модель | Val IoU | Val Accuracy | 
|--------|---------|--------------|
| U-Net (базовая) | 0.7233 | 0.7257 | 
| U-Net (skip connections) | 0.7300 | 0.7380 |
| PSPNet | 0.7661 | 0.7657 |

## 💡 Выводы
1. **Гиперпараметры** существенно влияют на качество: увеличение filters и kernel_size улучшает метрики до определенного предела
2. **Skip connections** в U-Net повышают точность на ~1% за счет сохранения пространственной информации
3. **PSPNet** с PPM показывает наилучший результат (Val IoU = 0.7661), особенно эффективен для сцен с объектами разных масштабов
4. **Оптимизация памяти** (uint8, int8, tf.data.Dataset) — необходимое условие работы с датасетами 1900+ изображений
